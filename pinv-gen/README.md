# pinv-gen

AI perfection and prompt engineering for PinV.

## Purpose
This package houses the system prompts, evaluation logic, and dataset analysis tools to bring widget generation to perfection.

## Features
- **Prompt Benchmarking:** Run the collected feedback dataset against new system prompts to measure improvement.
- **Few-Shot Library:** Curate high-quality examples to improve model reasoning.
- **LLM-as-a-Judge:** Automated scoring of generations based on technical and visual criteria.
